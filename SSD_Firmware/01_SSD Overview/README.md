### 1. HDD vs SSD   
<img src="assets/hdd and ssd img.png" width="550"/>

| **HDD의 내부구조** | **SSD의 내부구조** |
|---------------------|---------------------|
| Platters: 데이터를 저장하는 원판 (자기 디스크) | Controller: SSD의 두뇌, 데이터 흐름과 오류 제어 담당 |
| Spindle: 플래터를 회전시키는 축 | Cache (DRAM): 임시 데이터 저장 공간 |
| R/W Head: 데이터를 읽고 쓰는 장치 | NAND Flash Memory: 데이터를 실제로 저장하는 반도체 칩 |
| Actuator Arm: R/W 헤드를 이동시키는 팔 |  |
| Actuator: 팔을 제어하는 모터 |  |

#### 🔹HDD
HDD는 자기 디스크(플래터) 위에 데이터를 저장하는 저장장치다. 디스크가 고속으로 회전하며, 그 위를 이동하는 읽기/쓰기 헤드(Read/Write Head)가 데이터를 읽거나 쓴다. 이러한 구조는 전통적인 하드 드라이브로, 오랫동안 컴퓨터의 저장 장치로 널리 사용되어 왔다. 하지만 기계적인 움직임이 존재하기 때문에 물리적인 충격에 민감하고, 데이터 접근 속도가 비교적 느리다.
#### 🔹SSD
SSD는 반도체 칩(NAND Flash Memory)을 이용해 데이터를 저장하는 장치로, HDD와 달리 기계적인 움직임이 전혀 없다. 내부에는 컨트롤러, 캐시 메모리(DRAM), 그리고 여러 개의 NAND 플래시 메모리가 포함되어 있으며, 데이터를 전자적으로 처리한다. 이 덕분에 매우 빠른 속도, 높은 내구성, 낮은 전력 소비 등의 장점을 지닌다.

#### 🔸SSD와 HDD의 발전 과정과 동향
오랫동안 컴퓨터 저장장치의 주류는 HDD(Hard Disk Drive)였다. HDD는 회전하는 자기 디스크(플래터)에 데이터를 기록하고, 데이터를 읽고 쓰기 위해 읽기/쓰기 헤드가 물리적으로 이동하는 방식으로 동작한다. 이 구조는 비교적 저렴한 비용으로 대용량 저장이 가능하다는 장점이 있어, 2000년대까지 대부분의 데스크탑과 노트북, 서버 시스템에 널리 사용되었다.

그러나 HDD는 기계적인 부품으로 인해 속도가 느리고, 소음과 발열이 있으며, 외부 충격에 약하다는 단점도 존재했다. 이러한 한계는 사용자가 느끼는 성능 저하의 주요 원인이 되었고, 저장장치 기술의 전환 필요성을 불러왔다.

이러한 상황 속에서 등장한 SSD(Solid State Drive)는 저장 방식부터 완전히 다른 기술이다. SSD는 데이터를 NAND Flash Memory에 전자적으로 저장하며, 내부에 움직이는 부품이 없다. 이 덕분에 훨씬 빠른 속도, 낮은 전력 소모, 무소음, 높은 내구성이라는 강점을 가질 수 있었다. 초기 SSD는 매우 비싸고 용량이 작았기 때문에 고급 서버나 특수한 목적의 고성능 장비에만 제한적으로 사용되었지만, 시간이 흐르면서 여러 기술적 진보가 이루어졌다.

가장 큰 변화는 NAND 플래시 메모리의 발전이다. 초기에는 Cell 당 1비트를 저장하는 SLC(Single Level Cell) 방식이었으나, 이후 MLC(Multi Level Cell), TLC(Triple Level Cell), QLC(Quad Level Cell) 등으로 발전하며 저장 밀도는 증가하고 제조 단가는 감소했다. 또한 SSD의 컨트롤러 기술이 발전하면서 데이터 오류 복구, wear-leveling, garbage collection, Trim 명령어 지원 등 다양한 기능이 안정화되었고, SSD의 실사용 수명과 성능도 크게 향상되었다.

특히 NVMe(Non-Volatile Memory Express) 인터페이스의 등장은 SSD 기술 발전의 전환점이 되었다. 기존 SATA 인터페이스가 HDD에 맞춰 설계된 것이기 때문에 SSD의 성능을 제한하는 병목이 되었지만, NVMe는 PCIe 인터페이스를 기반으로 설계되어 훨씬 더 빠른 데이터 처리 속도를 제공한다. 덕분에 부팅 시간, 프로그램 실행, 파일 복사 등에서 SSD는 HDD보다 수배 이상 빠른 체감 성능을 제공할 수 있게 되었다.

시장 동향에서도 이러한 변화는 명확히 드러난다. Statista 등의 시장 분석 자료에 따르면 2023년 기준 노트북의 약 90% 이상은 SSD를 기본 저장장치로 탑재하고 있다. 또 다른 조사인 Trendfocus의 보고서에서는 2022년부터 PC용 SSD 출하량이 처음으로 HDD를 추월한 것으로 나타났다. 가격 측면에서도 SSD는 과거에 비해 큰 폭으로 하락했으며, 2012년에는 GB당 1달러 수준이었던 SSD의 가격이 2023년에는 0.07달러 수준까지 떨어졌다. 이는 HDD의 평균 단가인 GB당 0.03달러와 비교해도 격차가 크게 줄었음을 의미한다.

이러한 흐름 덕분에 SSD는 이제 개인용 PC뿐 아니라 서버, 클라우드, 모바일 디바이스, 게임 콘솔, 노트북 등 다양한 영역에서 기본 저장장치로 자리 잡고 있다. 빠른 속도와 뛰어난 전력 효율 덕분에 휴대성이 중요한 모바일 환경에서도 SSD는 매우 적합한 솔루션이다.

그렇다고 해서 HDD가 완전히 사라지는 것은 아니다. 여전히 HDD는 대용량 저장을 저렴한 비용으로 제공할 수 있다는 강점을 가지고 있으며, 백업 저장소, CCTV 저장 시스템, NAS(Network Attached Storage), 데이터 아카이빙 같은 용도로 꾸준히 활용되고 있다. 즉, 저장 공간의 경제성과 단순한 데이터 보존이 필요한 환경에서는 HDD의 역할이 여전히 유효하다.

결론적으로, 저장장치 시장은 현재 "성능은 SSD, 대용량 보관은 HDD"라는 역할 분담 구조로 재편되고 있다. SSD는 기술 발전과 가격 하락이라는 두 가지 요인을 바탕으로 시장을 빠르게 잠식하고 있으며, 앞으로도 HDD 대비 더 높은 채택률을 보일 것으로 예측된다. 특히 NVMe 기반 SSD는 데이터 중심 사회에서 고속 데이터 처리의 핵심 장치로 계속해서 중요성을 높여갈 것이다.
